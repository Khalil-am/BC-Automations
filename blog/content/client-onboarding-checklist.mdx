## Automation Name

**Client Onboarding Checklist Automation**

This automation generates a complete client onboarding package for new consulting engagements — including project kickoff checklists, stakeholder mapping templates, access request forms, and welcome communication templates. Ensures a professional and thorough start to every engagement.

---

## Prompt

```text
You are a senior engagement manager at a business consulting firm.
Generate a comprehensive client onboarding package for a new
[ENGAGEMENT TYPE] project with [CLIENT NAME].

The package should include:
1. Pre-Engagement Checklist — table with:
   - Task
   - Owner (Consultant/Client)
   - Due Date (relative: Day -5, Day -3, Day 0, etc.)
   - Status (Pending/Complete)
   - Notes
   Include tasks for: NDA, SOW sign-off, team introductions, access
   requests, environment setup, document sharing, tool provisioning
2. Stakeholder Map — table with:
   - Name (placeholder)
   - Role
   - Department
   - Influence Level (High/Medium/Low)
   - Interest Level (High/Medium/Low)
   - Communication Preference
   - Key Concerns
3. Kickoff Meeting Agenda
   - Meeting duration: 90 minutes
   - Agenda items with time allocations
   - Presenter for each item
   - Required pre-reads
   - Action items template
4. Welcome Email Template
   - Professional email from engagement lead to client team
   - Introduction of consulting team members
   - Project timeline highlights
   - Key milestones
   - Communication cadence
   - Shared workspace links (placeholders)
5. Access Request Matrix — table with:
   - System/Tool
   - Required Access Level
   - Requested For (role)
   - Approver
   - Status
   - Target Date
6. Week 1 Plan — daily breakdown with:
   - Day
   - Morning activities
   - Afternoon activities
   - Key meetings
   - Deliverables due
7. Communication Plan
   - Meeting type (standup, status, steering)
   - Frequency
   - Duration
   - Attendees
   - Medium (Teams/Zoom/In-person)
8. Risk & Assumptions Log (initial)
   - At least 5 common engagement risks
   - At least 5 standard assumptions

Generate a thorough onboarding package suitable for a mid-size
enterprise consulting engagement.
```

---

## Description

First impressions matter in consulting. The Client Onboarding Checklist Automation ensures that every new engagement starts with a structured, professional onboarding process. From pre-engagement preparation to the first week's detailed plan, this automation covers every aspect of client onboarding.

### Key Benefits

- **Professional First Impression** — Structured approach demonstrates consulting maturity
- **Nothing Falls Through** — Comprehensive checklist ensures every onboarding task is tracked
- **Stakeholder Alignment** — Built-in stakeholder mapping for effective communication planning
- **Repeatable Process** — Same quality onboarding experience for every client
- **Risk Mitigation** — Early identification of risks and assumptions

### How to Use

1. Copy the prompt above
2. Replace `[ENGAGEMENT TYPE]` (e.g., "ERP Implementation", "Digital Transformation")
3. Replace `[CLIENT NAME]` with the actual client name
4. Run the prompt in your preferred AI tool
5. Customize the checklist with project-specific requirements
6. Share relevant sections with the client team
7. Use the checklist to track onboarding progress during the first two weeks

### Onboarding Timeline

| Phase | Timeframe | Key Activities |
|-------|-----------|---------------|
| Pre-Engagement | Day -7 to Day -1 | NDA, SOW, access requests, environment setup |
| Kickoff | Day 0 | Kickoff meeting, team introductions, project overview |
| Discovery | Day 1 to Day 5 | Stakeholder interviews, system access, initial assessment |
| Stabilization | Day 6 to Day 14 | Regular cadence established, first deliverables |
