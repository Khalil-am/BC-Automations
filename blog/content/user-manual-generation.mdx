## Automation Name

**User Manual Generation Automation**

This automation streamlines the creation of professional user manuals for any business application, software platform, or internal tool. Instead of spending days writing documentation from scratch, use this prompt to generate structured, clear, and comprehensive user manuals in minutes.

---

## Prompt

```text
You are a senior technical writer specializing in business software documentation.
Generate a comprehensive user manual for [APPLICATION/SYSTEM NAME].

The manual should include:
1. Cover Page with document title, version, date, and company name
2. Table of Contents
3. Introduction & Purpose — explain what the system does and who it is for
4. System Requirements — hardware, software, and access prerequisites
5. Getting Started — login, first-time setup, and navigation overview
6. Feature-by-Feature Guide — each major feature with:
   - Feature name and description
   - Step-by-step instructions with numbered steps
   - Screenshots placeholders marked as [SCREENSHOT: description]
   - Tips and best practices
7. Troubleshooting — common issues and their solutions in a table format
8. FAQ Section — at least 10 frequently asked questions
9. Glossary — key terms and definitions
10. Contact & Support Information

Format the manual in a professional tone suitable for end users with
varying technical skill levels. Use clear headings, bullet points,
and numbered steps throughout.
```

## Description

The User Manual Generation Automation is designed for business consultants who need to deliver polished documentation as part of their client engagements. Whether you are onboarding a new ERP system, deploying a CRM platform, or rolling out an internal workflow tool, this automation produces a ready-to-edit user manual that covers every critical section.

### Key Benefits

- **Time Savings** — Reduce manual writing time from days to minutes
- **Consistency** — Every manual follows the same professional structure
- **Completeness** — Built-in checklist ensures no section is missed
- **Customizable** — Easy to adapt the prompt for any application or industry
- **Client-Ready** — Professional formatting suitable for direct delivery

### How to Use

1. Copy the prompt above
2. Replace `[APPLICATION/SYSTEM NAME]` with your target system
3. Run the prompt in your preferred AI tool
4. Review and customize the generated output
5. Add actual screenshots where placeholders are marked
6. Export as PDF or Word document for client delivery

### Best Practices

- Always review generated content for accuracy against the actual system
- Add real screenshots to replace placeholder markers
- Customize the troubleshooting section with known client-specific issues
- Include version numbers and revision dates for document control
