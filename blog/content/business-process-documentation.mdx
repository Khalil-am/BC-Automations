## Automation Name

**Business Process Documentation Automation**

This automation produces comprehensive business process documentation including process narratives, swimlane flow descriptions, RACI matrices, and Standard Operating Procedures (SOPs). Essential for consultants conducting process assessments, redesigns, or system implementation projects.

---

## Prompt

```text
You are a senior business process consultant with expertise in process
documentation and optimization.
Generate comprehensive business process documentation for
[PROCESS NAME] in [DEPARTMENT/ORGANIZATION].

The documentation should include:
1. Process Overview
   - Process name, ID, and version
   - Process owner and stakeholders
   - Purpose and objectives
   - Scope (in-scope and out-of-scope)
   - Key performance indicators (KPIs)
2. Process Narrative
   - Detailed step-by-step description of the process
   - Decision points and branching logic
   - Inputs and outputs for each step
   - Systems and tools used at each step
3. RACI Matrix — table with columns:
   - Activity/Step
   - Responsible (who does the work)
   - Accountable (who is ultimately answerable)
   - Consulted (who provides input)
   - Informed (who is kept updated)
4. Swimlane Flow Description
   - Each lane representing a role/department
   - Sequential steps with decision diamonds described
   - Handoff points between lanes clearly marked
5. Standard Operating Procedure (SOP)
   - Numbered steps with detailed instructions
   - Required forms and documents
   - System transactions/screens
   - Quality checkpoints
6. Exception Handling
   - Known exceptions and their resolution paths
   - Escalation procedures
   - Workaround instructions
7. Process Metrics
   - Cycle time targets
   - Volume expectations
   - Quality thresholds
8. Revision History table

Generate documentation for a process with at least 12 steps
and 3 decision points.
```

## Description

Business process documentation is the backbone of any consulting engagement. Whether you are mapping current-state processes, designing future-state workflows, or creating training materials, this automation delivers structured documentation that meets enterprise standards.

### Key Benefits

- **Enterprise-Grade** — RACI matrices, KPIs, and SOP formats used by top consulting firms
- **Complete Package** — Process narrative, flow description, and operating procedures in one document
- **Decision Logic** — Built-in handling of branching and exception scenarios
- **Measurable** — Includes KPIs and process metrics for performance tracking
- **Audit-Ready** — Revision history and version control built in

### How to Use

1. Copy the prompt above
2. Replace `[PROCESS NAME]` (e.g., "Procure-to-Pay", "Hire-to-Retire")
3. Replace `[DEPARTMENT/ORGANIZATION]` (e.g., "Finance Department", "HR Division")
4. Run the prompt in your preferred AI tool
5. Review against actual process observations and stakeholder interviews
6. Add organization-specific details, names, and system references
7. Create visual flowcharts based on the swimlane descriptions

### Process Documentation Hierarchy

| Level | Document | Audience | Detail Level |
|-------|----------|----------|-------------|
| L1 | Process Map | Executives | High-level overview |
| L2 | Process Narrative | Managers | Detailed steps and decisions |
| L3 | SOP | End Users | Step-by-step instructions |
| L4 | Work Instructions | Operators | Granular task details |
